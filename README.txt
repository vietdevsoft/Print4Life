PRINT4LIFE CENTERED HERO VERSION

Cách dùng:
1. Giải nén zip.
2. Tạo thư mục assets cùng cấp index.html.
3. Đặt video hero vào: assets/hero-3d.mp4
4. Có thể đặt thêm poster: assets/video-poster.jpg
5. Mở index.html bằng Chrome/Edge.

Bản này đã chỉnh:
- Font Be Vietnam Pro để hiển thị tiếng Việt tốt hơn.
- Hero dùng video làm background.
- Text giới thiệu dự án căn giữa.
- Overlay tối để chữ rõ.
